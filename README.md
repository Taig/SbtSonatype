# sbt Houserules Sonatype Plugin

[![CircleCI](https://circleci.com/gh/Taig/sbt-sonatype/tree/master.svg?style=shield)](https://circleci.com/gh/Taig/sbt-sonatype/tree/master)

## Installation

```scala
addSbtPlugin( "io.taig" % "sbt-sonatype" % "1.1.1" )
```
